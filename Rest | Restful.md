# Rest 와 Restful
## Rest
“Representational State Transfer” 의 약자이다.   
분산시스템 설계를 위한 소프트웨어 아키텍처의 하나로서 
웹에 존재하는 모든 자원(이미지, 동영상, DB 자원)에 고유한 URI를 부여해 활용하는 것으로, 자원을 정의하고 자원에 대한 주소를 지정하는 방법론을 의미한다고 한다.

## Rest 구성요소
* Resource (자원), URL

* Verb (행위), Method

* Representation (표현)
