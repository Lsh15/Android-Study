# Kotlinx Serialization
kotlinx Serialization은 JetBrains에서 만든, Kotlin을 위한 JSON 라이브러리이다.   
Serialization는 응용 프로그램에서 사용하는 데이터를 네트워크를 통해 전송하거나 데이터베이스 또는 파일에 저장할 수 있는 형식으로 변환하는 프로세스이다.


## 특징
*	빠른 JSON 인코딩 & 디코딩을 지원

* 멀티플랫폼을 지원

* 코틀린을 지향

* 컴파일 안전을 보장
